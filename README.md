A Python project using Pandas to explore 32,000+ horror films.

## What it does
- Filters out "noise" (movies with low vote counts).
- Identifies the most profitable horror films in the dataset.
- Exports a Top 10 list to a CSV file.

## Tech Used
- **Python**
- **Pandas** (for data manipulation)

## How to Run
1. Clone this repo.
2. Ensure you have the dataset in a `/data` folder.
3. Run `python analyzer.py`.
